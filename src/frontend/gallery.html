PASA provides both textual and graphical output. The textual output is cleaned and structured (to facilitate further ad-hoc analyses) and includes, for example, a list of curated peptides that were found to be enriched in the Elution with respect to the Flow-through, and a list of CDR3 Informative peptides (i.e., peptides that were mapped to antigen-specific antibodies). In addition to various textual outputs PASA provides several plots that summarize different statistics of the data. For example:

(1) Isotypes distribution



(2) A bar plot of the AA length of the human CDRH3 (the CDR3 region of the heavy chain)



(3) A bar plot showing the frequencies of the various IGHV family usage



(4) A bar plot showing the frequencies of the various DJ family usage in CDRH3



(5) A bar plot showing the frequencies of the various VDJ family usage in CDRH3



(6) A scatter plot showing the proteomic normalized intensity (per million) of peptides mapped to a user-provided BCR-Seq database intensity (per million). This plot is generated per database.
